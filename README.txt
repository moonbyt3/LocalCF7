== Description ==

This plugin saves CF7 submissions to the database where wordpress is installed. It makes separate table for submited data, saves contact form title, date and time when it's sent, and all data inside CF7 form


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Clone repository to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
